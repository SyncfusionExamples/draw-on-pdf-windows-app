# Draw on PDF using Windows App
This repository contains the project to draw on PDF document in the UWP application using [Syncfusion UWP PDF Viewer](https://www.syncfusion.com/uwp-ui-controls/pdf-viewer)

## How to run the project

* Checkout this project to a location in your disk.
* Open the solution file using the Visual Studio.
* If you are targetting to deploy this project to the local machine, then make sure that the Build Configuration is set accordingly.
* Run the project to install the required nuget packages and deploy the application.
